# 🚗 Smart Workshop Manager

A console-based Java CRUD application designed to manage vehicles and their service records efficiently.

---

## 📌 Features

- ➕ Add new vehicles  
- 📋 View all registered vehicles  
- ✏️ Update vehicle details  
- ❌ Delete vehicles  
- 🛠️ Manage service records  
- 💾 Persistent storage using file handling  
- 📊 Basic analytics dashboard  

---

## 🛠️ Tech Stack

- Java (Core)
- OOP Concepts
- File Handling (BufferedReader, FileWriter)
- Collections (HashMap, ArrayList)
